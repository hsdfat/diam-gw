# Basic Examples

Simple examples demonstrating basic Diameter protocol operations.

## Examples

### simple_cer_cea.go
Basic CER/CEA handshake example (legacy, predates client package).

## Running Examples

```bash
# Run basic CER/CEA example
go run simple_cer_cea.go
```

## Note

These are basic examples. For production use, see the `s13_client` example which uses the full-featured client package.
